# Netflix Movies Data Analysis
This project focuses on analyzing a Netflix Movies dataset using Python for data cleaning, preprocessing, exploratory data analysis (EDA), and visualization. The objective is to uncover meaningful insights about movie genres, ratings, popularity, and release trends. 
# Dataset Information
Total Records: 9827 Movies
Attributes: 9 Columns
Source: Netflix Movies Dataset
# Data Preprocessing
The following preprocessing steps were performed: 
Removed Unnecessary columns:
Overview
Original_Language
Poster_Url
Converted Release_Date into datetime format.
Extracted only the release year for analysis.
Standardized and cleaned Genre values.
Split multiple genres into individual categories.
Converted Vote_Average into rating categories:
Blockbuster
Well Received
Average Performer
Low Rated
Checked and removed duplicate records.
Verified missing values.
# Exploratory Data Analysis (EDA)
The project answers the following business questions:
# Q.1 : What is the most frequent genre of movies released on Netflix??
# Q.2 : Which movie has the highest vote average?
# Q.3 : Which movie has the highest popularity and what is its genre?
# Q.4 : Which movie has the lowest popularity and what is its genre?
# Q.5 : Which year has the highest number of movie releases?
